# Stock-Market-Trend-Prediction-using-ML

A machine learning project that predicts stock market trend (UP/DOWN) using historical stock data, technical indicators, and classification models such as Random Forest and XGBoost.

## Project Overview

This project uses historical stock market data to predict future stock movement based on previous market trends. It helps understand how machine learning can be applied in financial forecasting and time-series prediction.

## Features

- Real-time stock data using yFinance  
- Data preprocessing and feature engineering  
- Technical indicators like Moving Average, Returns, Volatility, RSI  
- Random Forest model  
- XGBoost model  
- Trend prediction (UP / DOWN)  
- Accuracy evaluation

## Technologies Used

Python, Pandas, NumPy, Matplotlib, Scikit-learn, XGBoost, yFinance, Google Colab.

## Dataset

Historical stock market data downloaded using yFinance (Example: TCS.NS).

## Workflow

1. Data Collection  
2. Data Cleaning  
3. Feature Engineering  
4. Train-Test Split  
5. Model Training  
6. Prediction  
7. Evaluation

## Models Used

- Random Forest Classifier  
- XGBoost Classifier

## Results

- Random Forest Accuracy: ~49%  
- XGBoost Accuracy: ~51%

Stock market prediction is highly complex and noisy, so even moderate improvements are meaningful.
